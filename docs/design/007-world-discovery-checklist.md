# Checklist de descubrimiento del mundo y NPCs

Documento de preguntas para definir el mundo y sus personajes de forma progresiva, con foco en interacciones NPC <-> jugador y NPC <-> NPC.

## 1) Punto de partida del jugador
- [ ] Por que el jugador llega aqui y no a otro lugar?
- [ ] Que cree el jugador que debe lograr al inicio?
- [ ] Que le falta para avanzar (info, objeto, permiso)?
- [ ] Que cree el jugador que esta en juego si falla?

## 2) Contexto del lugar
- [ ] Que es este lugar (funcion y simbolo)?
- [ ] Que reglas se aplican aqui y quien las impone?
- [ ] Que se ve primero y que se oculta?
- [ ] Que elementos del entorno dan pistas sin dialogo?

## 3) NPCs (definicion rapida)
Para cada NPC:
- [ ] Rol concreto en el lugar.
- [ ] Que quiere del jugador y que teme.
- [ ] Que sabe y que no sabe.
- [ ] Que ofrece y que bloquea.
- [ ] Que contradiccion interna lo define.

## 4) Relaciones entre NPCs
- [ ] Como se conocen y por que siguen aqui?
- [ ] Que se deben entre ellos (deuda, favor, culpa)?
- [ ] Que informacion comparten y cual ocultan?
- [ ] Hay conflicto directo o solo diferencias de metodo?
- [ ] Que haria que uno traicione al otro?

## 5) Interaccion NPC <-> jugador
- [ ] Como se gana confianza con cada NPC?
- [ ] Que acciones rompen la relacion?
- [ ] Que pide el NPC a cambio de ayudar?
- [ ] Que mentira del jugador si detecta y cual no?
- [ ] Que cambia si el jugador ya hizo algo grave?

## 6) Puerta y llave (objetivo central)
- [ ] Quien tiene la llave principal y por que?
- [ ] Cuantas llaves existen y para que sirven?
- [ ] Que hace el cerrojo y como se fuerza?
- [ ] Que vias existen para abrir y que costo moral tienen?
- [ ] Que via parece buena pero pesa en contra?
- [ ] Que via parece mala pero pesa a favor?

## 7) Progresion de informacion (descubrimiento)
- [ ] Que aprende el jugador primero?
- [ ] Quien le revela que hay una puerta?
- [ ] Como descubre que existe una llave?
- [ ] Como aprende las vias alternativas (robo, fuerza, manipulacion)?
- [ ] Que pistas ambientales refuerzan cada via?

## 8) Consecuencias y juicio
- [ ] Que consecuencia inmediata tiene cada via?
- [ ] Que peso moral deja cada via en el juicio?
- [ ] El jugador entiende el peso antes o despues?
- [ ] Que argumentos honestos puede dar el jugador al juez?

## 9) Memoria y persistencia
- [ ] Que eventos se guardan en memoria y con que tags?
- [ ] Que NPC recuerda que cosa y por que?
- [ ] Como se traduce la zona del evento en memoria?
- [ ] Que acciones son publicas vs privadas?

## 10) Ritmo y revelaciones
- [ ] Que misterios quedan abiertos tras cada encuentro?
- [ ] Que revelacion cambia la lectura de un NPC?
- [ ] En que momento el jugador entiende la dualidad moral?

## 11) Chequeo de coherencia
- [ ] Hay algo que un NPC sabe sin razon?
- [ ] Hay una via que contradice motivaciones previas?
- [ ] El peso moral coincide con la intencion del tono?
- [ ] El juez es consistente con su omnisciencia?

## 12) Cierre de arco
- [ ] Que cambia cuando la puerta se abre?
- [ ] Que queda abierto para el jugador tras el juicio?
- [ ] Que aprende el jugador sobre su propia agencia?
