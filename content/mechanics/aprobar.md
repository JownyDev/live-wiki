# Aprobar

**Aprobar** es la condición de éxito de un proyecto bajo el sistema de [Nota](mechanic:nota).

Cuando un proyecto cumple con los requisitos mínimos establecidos y alcanza una puntuación suficiente, se considera aprobado. Esto suele desbloquear nuevos recursos, reconocimiento o la siguiente fase del desarrollo.
